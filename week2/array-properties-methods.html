Array Properties and Methods

In JavaScript, Arrays are powerful and super useful. Arrays have some very useful inbuilt methods that are demonstrated in the following examples.

Given the following array:

const fruits = ['bananas 🍌', 'oranges 🍊', 'mangoes 🥭', 'strawberries 🍓', 'watermelons 🍉'];

To find the length of an array, we can use the length property:

fruits.length;
>> 5

You can also find the last item in the array this way:

fruits[fruits.length - 1];
>> 'watermelons'

The reason why we had to subtract 1 from the length of the array is because of the way the array items are indexed. The first index starts at 0, so the last item will have an index of one less than the array's length.

The length property of an array is mutable. if you make the array longer, the extra slots will be filled in with undefined values, whereas if you make the array shorter, the extra elements will be removed completely! 

// Making the array longer
fruits.length = 7
>> 7;

fruits
>> ['bananas 🍌', 'oranges 🍊', 'mangoes 🥭', 'strawberries 🍓', 'watermelons 🍉', undefined, undefined];

// Making the array shorter, truncating the elements.
fruits.length = 3
>> 3

fruits
>> ['bananas', 'oranges', 'mangoes']

You can also truncate an array using Array splice() method: 

const fruits = ['bananas 🍌', 'oranges 🍊', 'mangoes 🥭', 'strawberries 🍓', 'watermelons 🍉'];

// Truncate the array, removing elements after the 3rd index
fruits.splice(3);

fruits
>> ['bananas 🍌', 'oranges 🍊', 'mangoes 🥭']

We use the pop() method to remove the last item in the array, and the shift() method to remove the first item in the array. Both methods return the value that was removed.

fruits;
>> ['bananas 🍌', 'oranges 🍊', 'mangoes 🥭']

// Remove last element
fruits.pop();
>> 'mangoes 🥭'

// Remove first element
fruits.shift();
>> 'bananas 🍌'

fruits;
['oranges 🍊']

The push() method is used to append an item to the end of an array, and the unshift() method appends an item to the beginning or start of the array. The return value for both methods is the length of the array.

// Append an element to the end of the array
fruits.push('coconuts 🥥');
>> 2

// Append an element to the beginning of the array
fruits.unshift('pineapples 🍍');
>> 3

fruits;
>> ['coconuts 🥥', 'oranges 🍊', 'pineapples 🍍'

The concat() method is used to merge an array with one or more arrays: 

fruits.concat(['tomatoes 🍅', 'grapes 🍇', 'melons 🍈', 'peaches 🍑'], ['pears 🍐', 'apples 🍎']);
>> ['coconuts 🥥', 'oranges 🍊', 'pineapples 🍍', 'tomatoes 🍅', 'grapes 🍇', 'melons 🍈', 'peaches 🍑', 'pears 🍐', 'apples 🍎']

Note that this doesn't change the fruits array, instead it simply creates an array by combining the three arrays. Another way to merge arrays is to use the spread operator that was added to ES6. The spread operator is written thus ...

const newfruits = [...fruits, ...['tomatoes 🍅', 'grapes 🍇', 'melons 🍈', 'peaches 🍑']];
>> ['coconuts 🥥', 'oranges 🍊', 'pineapples 🍍', 'tomatoes 🍅', 'grapes 🍇', 'melons 🍈', 'peaches 🍑']

The join() method is used to convert an array into a string that comprises all the elements in the array, separated by commas by default. You can also specify another separator by passing it as an argument. 

fruits.join();
>> 'coconuts 🥥, oranges 🍊, pineapples 🍍, tomatoes 🍅, grapes 🍇, melons 🍈, peaches 🍑'

// you can specify the separator too
fruits.join(' + ');
>> 'coconuts 🥥 + oranges 🍊 + pineapples 🍍 + tomatoes 🍅 + grapes 🍇 + melons 🍈 + peaches 🍑'

The slice() method creates a subarray by chopping out a slice of an original array, starting at one position and finishing at another.

/*
starts at the 3rd item (index 2) and finishes at the fourth item (the item with index 4 is not included)
*/
fruits.slice(2, 4);
>> ['pineapples 🍍', 'tomatoes 🍅', 'grapes 🍇']

The splice() method can remove items from an array and then insert new ones in their place. For example, the following code removes 'tomatoes 🍅' and replaces it with 'bananas 🍌' . Be aware that this is a destructive operation as it changes the original array.

fruits.splice(1, 1, 'bananas 🍌');
>> ['pineapples 🍍', 'bananas 🍌', 'grapes 🍇']

The first parameter is the index to start the splice, the second parameter indicates how many items should be removed from the array, every value passed after the second parameter is then inserted in the index specified in the first parameter.

The splice() method can also insert values into an array at a specific index without removing any items, by indicating that zero items are to be removed. This will return an empty array (because nothing was removed). 

fruits.splice(2, 0, 'tomatoes 🍅');
>> []

fruits;
>> ['pineapples 🍍', 'bananas 🍌', 'grapes 🍇', 'tomatoes 🍅']

You can also use the splice() method to remove an item from an array. This is demonstrated below:

// Delete an item at the 3rd position (index 2) in the array.
fruits.splice(2, 1); // note that no values were added as parameters
>> ['grapes 🍇']

fruits;
>> ['pineapples 🍍', 'bananas 🍌', 'tomatoes 🍅']

We can reverse the order of an array using the reverse() method: 

fruits.reverse();
>> ['tomatoes 🍅', 'bananas 🍌', 'pineapples 🍍']

[1, 2, 3].reverse();
>> [3, 2, 1]

We can sort the order of an array using the sort() method:

// Sort alphabetically by default
fruits.sort();
>> ['bananas 🍌', 'pineapples 🍍', 'tomatoes 🍅']

You can pass a callback method to the sort() method when sorting an array:

[2, 1, 3].sort((a, b) => a - b);
>> [1, 2, 3]

The indexOf() method finds the index of an item in an array. If the item does not exist it returns -1 .

const fruits = ['pineapples 🍍', 'bananas 🍌', 'grapes 🍇', 'tomatoes 🍅']

fruits.indexOf('grapes 🍇');
>> 2

// returns -1 if item does not exist
fruits.indexOf('coconut 🥥');
>> -1

ES6 introduced the includes() method which returns a Boolean value if an item is found in an array.

fruits.includes('pineapples 🍍');
>> true

fruits.includes('coconut 🥥');
>> false
